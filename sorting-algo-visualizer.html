Sorting_Algo_Visualizer

This project is a console-based application in Java that visually represents the execution of various sorting algorithms on integer arrays. It includes Merge Sort, Bubble Sort, Heap Sort, Selection Sort, and Insertion Sort.

Key Features

1. User Interaction through Console
Users can select a sorting algorithm to visualize and choose between generating a random array, entering a custom array, or using a previously generated array. The system shows each sorting step and displays the number of iterations.
2. Sorting Algorithms
The visualizer includes implementations of:
- Merge Sort
- Bubble Sort
- Heap Sort
- Selection Sort
- Insertion Sort
3. Visualization
The VisualizerAlgo class represents numbers as columns of '*' characters. Each number's value determines the column's height, creating a clear visual of the sorting progress. The number of iterations is counted and displayed.
4. Array Generation and Handling
Users can generate random arrays with values between 0 and 100, create custom arrays, and copy or display existing arrays for sorting.
5. Error Handling
The system validates inputs and handles exceptions, such as when attempting to find the largest number in an empty or null array.
Conclusion
This Sorting Algorithm Visualizer helps users understand how popular sorting algorithms work by providing a step-by-step visual representation in the console, making algorithm complexity more approachable.
