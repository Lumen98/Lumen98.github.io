Falling Sand Game

This Falling Sand Game is a Java-based console application designed to visually simulate the behavior of various particles, such as sand, water, and ice. The application provides a graphical representation of these particles and their interactions in a grid-based environment.

Key Features

1. User Interaction through GUI

The application utilizes Javaâ€™s AWT and Swing libraries to create an interactive graphical interface. Users can select different particle types and place them on a grid. The application displays the movement and interactions of these particles in real time.

2. Particle Simulation

The system supports various particle types, including:

Empty (Eraser)

Metal

Water

Sand

Ice

Glass

Each particle type has its unique behavior and interactions with other particles. For example, sand particles fall downwards until they hit another particle, and ice particles can melt into water over time.

3. Visualization

The SandDisplay class is responsible for the visualization of the grid and particles. Each cell in the grid can contain a particle, and the color of the cell represents the type of particle it contains. The application updates and repaints the display in real time to show particle movements and interactions.

4. User Controls

Users can interact with the application using mouse clicks to place particles on the grid. A slider allows users to adjust the simulation speed. Buttons enable users to select the type of particle they want to place.

5. Randomized Particle Movement

The step() method in the SandLab class is responsible for simulating the movement and interactions of particles. Particle movements are randomized to create a dynamic and unpredictable simulation.

6. Error Handling and Robustness

The system includes checks to ensure particles do not move outside the grid boundaries. Interactions between different particle types, such as sand and water, are handled to ensure realistic behavior.

Conclusion

This Falling Sand Game offers a captivating visual representation of particle behavior and interactions. By allowing users to interactively place and observe particles, the application provides insights into the dynamics of particle systems. Whether used for educational purposes or sheer entertainment, this application serves as a testament to the power of simulation and visualization in understanding complex systems.
