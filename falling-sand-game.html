Falling Sand Game

This Falling Sand Game is a Java-based console application designed to visually simulate the behavior of various particles such as sand, water, and ice. The application provides a graphical representation of these particles and their interactions in a grid-based environment.

Key Features

1. User Interaction through GUI
The application uses Javaâ€™s AWT and Swing libraries to create an interactive interface. Users can select different particle types and place them on a grid, watching their movement and interactions in real time.
2. Particle Simulation
Supports particles like Empty (Eraser), Metal, Water, Sand, Ice, and Glass, each with unique behaviors. For instance, sand falls until it hits another particle, and ice can melt into water over time.
3. Visualization
The SandDisplay class manages the grid visualization. Each cell represents a particle type with a corresponding color, updating in real time as particles move and interact.
4. User Controls
Users can place particles with mouse clicks, adjust simulation speed via a slider, and select particle types with buttons.
5. Randomized Particle Movement
The step() method in the SandLab class introduces randomized behavior to simulate dynamic particle movement and interactions.
6. Error Handling and Robustness
The system prevents particles from moving outside the grid and handles interactions like sand and water mixing, ensuring smooth and realistic simulation behavior.
Conclusion
This Falling Sand Game offers an engaging visual simulation of particle dynamics. By allowing users to place and observe various particle types, it provides both educational insights and an entertaining sandbox environment.
