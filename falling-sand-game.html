Falling Sand Game

This Falling Sand Game is a Java-based console application designed to visually simulate the behavior of various particles, such as sand, water, and ice. The application provides a graphical representation of these particles and their interactions in a grid-based environment.

Key Features

1. User Interaction through GUI
The application utilizes Java’s AWT and Swing libraries to create an interactive graphical interface. Users can select different particle types and place them on a grid. The application displays the movement and interactions of these particles in real time.

2. Particle Simulation
The system supports various particle types, including Empty (Eraser), Metal, Water, Sand, Ice, and Glass. Each has unique behaviors and interactions. For example, sand falls downward until it hits another particle, and ice can melt into water over time.

3. Visualization
The SandDisplay class is responsible for the visualization of the grid and particles. Each cell can contain a particle, and the cell’s color reflects the type of particle. The application updates and repaints in real time to show movement and interactions.

4. User Controls
Users can place particles on the grid with mouse clicks. A slider lets them adjust simulation speed. Buttons allow them to select the particle type to place.

5. Randomized Particle Movement
The step() method in the SandLab class simulates particle movements. Randomization adds dynamic, unpredictable behavior to the simulation.

6. Error Handling and Robustness
The application prevents particles from moving outside the grid and properly handles interactions like sand and water mixing. These checks ensure realistic behavior without errors.

Conclusion
This Falling Sand Game provides a captivating visual representation of particle behaviors. By allowing users to place and observe different particle types, it offers educational insight into particle dynamics and an entertaining simulation environment.
