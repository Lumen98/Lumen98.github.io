Snake_Classic

Snake is one of those instantly recognizable games for everyone. It brings back a nostalgia that is unreplicable. I wanted to remake this game but with my own custom features. This project is a rendition of the classic Snake game developed using the LibGDX framework. The game is played in a grid where the snake moves around, consuming apples to grow in size while avoiding collisions with itself, rocks, and rotten apples (hint: those are the new features).

Key Features

1. Game Initialization
The game is launched using the GameLauncher class. Screen dimensions are defined through constants, and the game instance is initialized through the Snake class.
2. Game Components
Snake: Includes a head (SnakeHead) and body (SnakePart).
Apple: Increases snake size when consumed.
Rock: Causes a game over if hit.
Rotten Apples: Resets the snake's body and score when eaten.
3. Game Mechanics
The snake moves with arrow key inputs. Collision detection manages interactions with apples, rocks, rotten apples, the snake's body, and game boundaries. Scores and high scores are tracked and saved using Preferences.
4. Graphics and Visualization
The game utilizes ShapeRenderer for rendering, with a texture (snakeHeadPic) for the snake's head. A simple menu features a “START!” button, and current and high scores are displayed during gameplay.
5. Game Flow
The game begins with a menu screen. Upon pressing “START!”, gameplay starts. The game ends when the snake collides with its body, a rock, or the boundary, after which the player is returned to the menu screen.
6. Utility Classes
Constants: Stores dimension and grid-related constants.
Direction: Enum representing possible snake movement directions.
