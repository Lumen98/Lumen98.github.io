Snake Classic

Snake is one of those instantly recognizable games for everyone. It brings back a nostalgia that is unreplicable. I wanted to remake this game but with my own custom features. The project is a rendition of the classic Snake game developed using the LibGDX framework. The game is designed to be played in a grid where the snake moves around, consuming apples to grow in size, while avoiding collisions with itself, rocks, and rotten apples (hint: those are the new features).

Key Features

1. Game Initialization
The game is launched using the GameLauncher class. Screen dimensions are set using constants for width and height. The game instance is created via the Snake class.

2. Game Components
Snake: Represented by a head (SnakeHead) and body (SnakePart). Apple: When consumed, the snake grows in size. Rock: Colliding with one ends the game. Rotten Apples: Consuming these clears the snake’s body and resets the score.

3. Game Mechanics
The snake moves based on arrow keys. The system checks for collisions with apples, rocks, rotten apples, the snake’s body, and boundaries. Player score and high score are tracked, saved, and updated using Preferences.

4. Graphics and Visualization
The game uses ShapeRenderer for most drawing, with a texture image (snakeHeadPic) for the snake’s head. There’s a simple menu screen with a “START!” button, plus on-screen display of current and high scores.

5. Game Flow
The game starts in a menu screen. Upon “START!”, it transitions to gameplay. The game ends when the snake collides with its own body, a rock, or the boundary. Afterwards, the player is returned to the menu.

6. Utility Classes
Constants: Holds dimension/grid constants. Direction: An enum representing potential snake movement directions.
