FPS_Booster

I used to play video games a ton, and when I did, I played on a super low end PC. I would always have to terminate the same Microsoft/Windows bloatware processes to squeeze out as much FPS as I could. I ended up making FPSBooster which automates this process by acting as a Task Manager, but with the option to kill all the processes I add to a list with the click of a button.

Key Features

1. Modular GUI Architecture
Designed and implemented a scalable, multi-tabbed GUI using PyQt5, separating Basic, Advanced, and Manage Lists views to promote maintainability and extensibility.

2. Real-Time System Resource Monitoring
Integrated Psutil to track and display CPU, Memory, and GPU usage across active processes, with rolling average calculations for consistent performance insights.

3. Automated Process Optimization
Engineered a One-Click Boost system to safely terminate blacklisted processes while ensuring critical system stability, following best practices for process management.

4. Dynamic Whitelist and Blacklist Management
Built a persistent JSON-backed system for user-defined whitelist/blacklist control, providing fine-grained management of process behavior within the application.

5. Safe Termination with System Integrity Checks
Implemented safeguards to detect and prevent termination of essential Windows system processes, including administrative privilege verification and fail-safes.

6. Data Handling and Caching
Developed efficient local caching mechanisms to store user preferences and previous process states, minimizing redundant system calls and improving runtime performance.

7. Community Engagement and Validation
Open-sourced on GitHub achieving over 23,000+ views across different social media platforms, demonstrating strong community traction, feedback-driven iterations, and collaborative development.
